# 🛒 Market Basket Analysis

This project performs **Market Basket Analysis** using data mining techniques to discover relationships between products.

## 📊 Algorithms Used
- Apriori Algorithm
- FP-Growth Algorithm

## 💡 Features
- Association Rule Mining
- Rule Filtering (Support, Confidence, Lift)
- Interactive Dashboard using Streamlit
- Data Visualization (Histogram, Scatter Plot, Bar Charts)
- 🌍 Country-wise Transaction Analysis
- 🛒 Top Products by Country
- Download Filtered Results

## 🛠️ Tech Stack
- Python
- Pandas
- Streamlit
- Plotly

## 📁 Dataset
- Online Retail Dataset (Excel format)

pip install -r requirements.txt
